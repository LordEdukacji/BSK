Detailed Description

Generates the RSA key pair.

The public key is available directly in its byte form, while the private key is secured using a PIN provided during the generation process.

Generates a pair of keys and saves them in the class fields.

Uses the RSA algorithm with a keysize of 4096. The PIN is hashed with SHA256 and the private key encrypted with AES.

pinPIN used for securing the private key.
